## Description

Frontend Application for task management.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# builds to production mode
$ npm run build

# The application should run by default at http://localhost:3001
# You can change the port at .env file
```

## Test

```bash
# unit tests
$ npm test
```
